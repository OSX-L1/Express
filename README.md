# Express
โปรแกรมสร้างใบปะหน้าจัดส่งสินค้าอัตโนมัติ (โหลดฐานข้อมูลจาก GitHub)

โปรเจกต์นี้คือเว็บแอปพลิเคชันแบบ Single HTML File ที่สร้างใบปะหน้าสำหรับจัดส่งสินค้า โดยมีคุณสมบัติเด่นคือการดึงและแคชฐานข้อมูลรหัสไปรษณีย์ของประเทศไทย (7,000+ รายการ) จากไฟล์ CSV ที่เก็บไว้บน GitHub โดยตรง

คุณสมบัติหลัก

โหลดข้อมูลรหัสไปรษณีย์อัตโนมัติ: ใช้ Raw URL ของไฟล์ CSV ที่มีข้อมูล จังหวัด/อำเภอ/ตำบล/รหัสไปรษณีย์

การแคชถาวร (LocalStorage): เมื่อโหลดข้อมูลสำเร็จแล้ว ครั้งต่อไปจะดึงข้อมูลจาก LocalStorage ในเบราว์เซอร์ทันทีเพื่อความรวดเร็ว โดยไม่จำเป็นต้องโหลดซ้ำจากอินเทอร์เน็ต

สร้างใบปะหน้า: กรอกข้อมูลผู้ส่งและผู้รับ แล้วสร้างใบปะหน้าสำหรับพิมพ์/บันทึกเป็น PDF ได้ทันที

Responsive: ใช้งานได้ดีทั้งบนคอมพิวเตอร์และมือถือ

Tech Stack: HTML, Tailwind CSS, Pure JavaScript.

วิธีการติดตั้งและใช้งาน (Deploy บน Netlify)

1. เตรียมไฟล์ CSV ของคุณ

ก่อนอื่น คุณต้องอัปโหลดไฟล์ CSV ที่มีข้อมูลรหัสไปรษณีย์ 7,000+ รายการของคุณไว้ใน Repository นี้ก่อน (แนะนำให้ใส่ในโฟลเดอร์ชื่อ data/)

2. คัดลอก Raw URL

หลังจากอัปโหลดไฟล์ CSV บน GitHub แล้ว ให้ทำตามขั้นตอนเหล่านี้เพื่อรับ Raw URL:

ไปที่ไฟล์ CSV ของคุณบน GitHub.

คลิกปุ่ม "Raw"

คัดลอก URL ที่อยู่บน Address Bar ทั้งหมด

3. แก้ไขโค้ด index.html

เปิดไฟล์ index.html แล้วแก้ไขค่าคงที่ DEFAULT_CSV_URL (อยู่ประมาณบรรทัดที่ 46) โดยแทนที่ด้วย Raw URL ที่คุณคัดลอกมา:

// *** CONFIGURATION: ใส่ Raw URL ของไฟล์ CSV ที่คุณอัปโหลดบน GitHub ที่นี่ ***
const DEFAULT_CSV_URL = "URL_ที่คุณคัดลอกมา"; 


4. Deploy ไปที่ Netlify

สร้าง Repository ใหม่บน GitHub แล้ว Commit ไฟล์ทั้งหมด (index.html, README.md, และไฟล์ CSV ของคุณ) เข้าไป

ไปที่ Netlify.com แล้วเข้าสู่ระบบ

คลิก "Add new site" และเลือก "Import an existing project"

เลือก GitHub เป็นผู้ให้บริการ Git และค้นหา Repository ที่คุณเพิ่งสร้าง

ในขั้นตอน "Deploy settings":

Branch to deploy: main (หรือชื่อ Branch หลักของคุณ)

Base directory: (เว้นว่างไว้)

Build command: (เว้นว่างไว้)

Publish directory: (เว้นว่างไว้)

คลิก "Deploy site"

Netlify จะทำการ Deploy เว็บไซต์ของคุณทันที และคุณจะได้ URL สำหรับใช้งานแอปพลิเคชันนี้!
